# 📦 Channel Common

A shared npm package containing reusable Zod validation schemas and TypeScript types for both the frontend and backend, ensuring end-to-end type safety and consistent validation.

## ✨ Features

- ✅ Shared Zod Schemas
- 🛡️ Centralized Input Validation
- 🏷️ Shared TypeScript Types
- 📦 Published as an npm Package
- 🔄 Reused Across Frontend & Backend

## 🛠️ Tech Stack

- TypeScript
- Zod
- npm

## 📦 Installation

```bash
npm install @sheelansu/channel-common
```

## 🚀 Usage

- Use **Zod schemas** for request validation in the backend.
- Import **shared types** in the frontend for type-safe API integration.
