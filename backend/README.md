# 🚀 Channel Blog Backend

A serverless REST API for the Channel Blog application, built with Hono and deployed on Cloudflare Workers.

## ✨ Features

- 🔐 User Authentication
- 📝 Blog CRUD APIs
- ✅ Request Validation with Zod
- 🗄️ Prisma ORM
- ⚡ Prisma Accelerate
- ☁️ Cloudflare Workers Deployment

## 🛠️ Tech Stack

- Hono
- Cloudflare Workers
- Prisma ORM
- Prisma Accelerate
- Zod
- bcryptjs
- JavaScript

## 📦 Installation

```bash
git clone https://github.com/Sheelansu/channel-blog.git
cd backend
npm install
```

## ▶️ Run Locally

```bash
npm run dev
```

## 🚀 Deploy

```bash
npm run deploy
```
