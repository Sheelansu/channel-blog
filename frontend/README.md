# 📺 Channel Blog

A modern React blog application with secure user authentication and a clean, responsive interface.

## 🚀 Live Demo

🔗 https://channel-blog-cs47aewrr-sheelansudev.vercel.app/

## ✨ Features

- 🔐 User Authentication
- 📝 Create & Read Blogs
- 📱 Responsive Design
- ⚡ Fast React Performance
- 🎨 Modern UI

## 🛠️ Tech Stack

- React.js
- React Router
- JavaScript
- CSS / Tailwind CSS
- Vercel

## 📦 Installation

```bash
git clone https://github.com/Sheelansu/channel-blog.git
cd frontend
npm install
npm run dev
```
